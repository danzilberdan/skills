# Personal Agent Skills

This repository contains personal agent skills.

It is usually located in `~/.agents/skills`.

Create skills at the root of this repository by default. Use nested folders when a clear category applies, such as `engineering/`.

Each skill should live in its own directory and include a `SKILL.md` file.

When creating or updating skills, use the Agent Skills best-practices guide:

- LLM-friendly Markdown: https://agentskills.io/skill-creation/best-practices.md
- Documentation index: https://agentskills.io/llms.txt
